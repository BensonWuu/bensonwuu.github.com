phpstrom快捷使用

  • ide使用

一些快捷键和操作，整理下来，有些不常用忘了可以看看，其实也是刚考完试休闲整理下，除特别说明外都是window下ps的操作

phpstorm快捷操作

Ctrl系列

  • ctrl + shift + n（常用）：查找文件，可输入文件名直接搜索全局匹配

  • ctrl + e（常用）：显示最近使用过的文件，可用于快捷跳转文件，解放鼠标

  • ctrl + shift + f：全局查找匹配输入的内容，个人相比其他用的最多

  • ctrl + 点击方法（变量名字）：直接跳转到相应方法下（变量声明处）

  • shift 按两下：全局模糊查找

  • ctrl+ R：替换

  • ctrl+shift+alt+n 便捷查找函数

  • ctrl + d：复制光标所在行到下一行快捷键

  • ctrl + y：删除当前行

  • ctrl + c/x ：复制、剪切当前行

  • ctrl + 左右方向键 ：间隔一个词以更快地移动光标

  • ctrl + shift + 左右方向键 ：可以更快选中你想选中的代码

  • ctrl + shift + 上下方向键 ：上下移动当前光标所在行

  • ctrl + shift + v：查看复制粘贴历史记录

  • ctrl + shift + u：选中后，切换字母大小写

  • ctrl + g ：跳到某一行

  • ctrl + n：查找一个类，可以输入模糊的命名空间

  • ctrl + n后输入类名字：加上行数，直接打开类并跳到某一行

  • ctrl + shift + m ：对于配对的括号（）和{}，[]等使用，跳到括号匹配的另一边

  • ctrl + p ：对带有参数的方法使用，可看到需要参数的信息

  • ctrl + q：对函数名使用，可以查看该函数相关的文档信息，对php内置函数还可以通过给的文档链接跳转查看用法，方便的一比（或者直接shift+F1直接跳转到文档）
    （注：Mac上使用F1查看文档）

  • ** Ctrl + F12（常用）**： 输入要跳转到的方法或者属性名

Alt系列

  • Alt Enter 大法好

在一个函数中直接Alt + Enter可以添加函数的一般注释

对array使用Alter + Enter ，可以变换新旧写法

对含有需要转义的字符使用Alt + Enter

  • Alt + j（多点编辑）：替换变量，统一修改变量名
  • Shift + Alt + Ctrl + j：全部选中
  • Shift + Alt + j：可以去掉最后添加的一个光标

（注：mac上是ctrl + g 和 shift + ctrl +g）

  • 按住Alt 并拖动鼠标选择，可以多个光标同时使用，进行统一输入或删除

  • Alt + s：分屏快捷键

  • Alt + insert：快捷设置getter setter方法等

  • Alt + 上下方向键 ：上下切换函数

  • Alt + F12：快捷弹出terminal

  • Alt + 1：快捷展开收缩侧边项目文件夹栏（mac下为cmd + 1）

其他

  • 点击文件侧边栏自动找到相应目录位置快捷设置

  • 手动找到文件目录位置

  • 设置是否显示面包屑，在ps的设置中搜索找到show breadcrumbs这个设置，看不惯就不选

  • 设置文件夹侧边栏字体

  • 设置侧边栏颜色与主代码界面颜色一致

  • 自定义php方法注释
