Apache Options相关配置字段

为了巩固下基础，还是不偷懒总结下这些基本的配置及其作用！学无止境

Options 选项

Options这个子句用来说明一些主要的设置，目前可以使用的设置有Indexes，Includes，FollowSymLinks，ExecCGI，
MultiView，当然还有两个最简单的选择就是None和All。

indexes：

设置在目录下没有index.html(或index.php）情况下是否允许显示目录

MultiViews：

（官网上Content negotiated "MultiViews" are allowed using mod_negotiation.）

FollowSymLinks：

（参考配置虚拟目录配置）决定是否可以通过符号链接跨越DocumentRoot。例如，尽管/ftp 不在
/home/httpd/html下面，但是我们仍然可以使用符号链接（软链接）建立一个/home/httpd/html/ftp
使得可以直接输入http://mydomain.com/ftp来访问这个目录。

使用FollowSymLinks 的办法很简单，就是首先在合适的目录段落里面Options followSymLinks，
（符号连接的上层就可以） 然后建立一个别名：Alias /ftp/“home/httpd/html/ftp/”后面的是
你建立的到/ftp 的符号链接。注意这一行应该位于所有目录段落之外。

AllowOverride (参数)：

是否允许各个目录用目录中的.htaccess(后面解释)覆盖这里设定的Options，它的选择有Options，
FileInfo，AuthConfig，Limit 或者它们的组合，当然还有None 和All。

Require ：

Order allow,deny Allow from all

Order allow,deny 的allow写在前面，deny写后面，故先看allow部分，Allow from all 是表示允许
访问所有文件，然后再看deny禁止访问的文件，这里deny没有写，所以没有文件被禁止访问。
