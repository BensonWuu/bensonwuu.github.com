  • git

什么是GitHub Pages，用来做什么？今天弄hexo搞定后，部署上github时候遇到点问题，解决后写下文章娱乐下啊哈哈~~

什么是GitHub Pages，用来做什么？

官方推出时候是这样说的：

使用GitHub Pages，自由做出自己想要的页面，一个大部分人喜欢的用法是写上自己blog，这里我结合hexo也弄上了自己的blog

创建过程

1、 在github上建立一个仓库，不过注意，仓库名字为username.github.io 或 username.github.com，以此名字建立完仓库后，在这里可以github已经默认为你生成一个可供访问的站点（如果不是放hexo博客的话，可以自己配置用github提供的页面）

这里我已经把hexo整体都搞定后部署上github了，直接bensonwuu.github.io可以访问到我的博客页面

2、 若不想总是以github提供的域名访问你的个人博客（我的是bensonwuu.github.io），比如说我想装逼点弄成blog.bensonwu.cn才访问到我的博客，可以设置下子域名访问,前提是你已经有了自己的域名可以设置相应的子域名，github貌似不支持在提供给你的这个域名下设置子域名

  • 方法是：在仓库根目录下创建了CNAME文件（全部大写），在文件中写上自己想要的子域名，如
    blog.bensonwu.cn（注意该文件中只能写一个域名）

  • 在自己已有的域名上配置子域名，选择CNAME方式，记录值写上github上提供的可直接访问你博客的地址
